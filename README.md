# Exploratory Data Analysis (EDA)

## Project Overview

This project was completed as part of a Data Analytics Internship. It focuses on Exploratory Data Analysis (EDA), SQL queries, data visualization, and dashboard creation to generate business insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* SQL
* Excel / PowerPoint

## Tasks Performed

* Performed descriptive statistics and univariate analysis.
* Created histograms, bar charts, scatter plots, and a correlation heatmap.
* Wrote SQL queries to answer business questions.
* Designed a static dashboard with key business KPIs.

## Deliverables

* EDA Report
* SQL Queries
* Dashboard Mock-up
* README
